# Notice Board App

A simple public notice board application built using Ruby on Rails and PostgreSQL.

## Features

- Create notices
- View notices
- Delete notices
- Pinned notices
- Category filtering
- Bootstrap UI
- Validations
- Seed data
- Model tests

---

## Tech Stack

- Ruby on Rails
- PostgreSQL
- Bootstrap

---

## Setup Instructions

1. Clone repository

```bash
git clone <repo-link>
```

2. Install gems

```bash
bundle install
```

3. Create database

```bash
rails db:create
```

4. Run migrations

```bash
rails db:migrate
```

5. Seed sample data

```bash
rails db:seed
```

6. Start server

```bash
rails server
```

Open:
http://localhost:3000/notices
